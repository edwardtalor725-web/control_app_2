# Финансовый контроль для малого бизнеса

## Описание
Веб-приложение для учета финансовых операций малого бизнеса с мультипользовательским доступом.

## Функционал
1. **Учет операций**
   - Внесение доходов и расходов с категориями
   - Привязка к контрагентам (клиенты, поставщики)
   - Платежный календарь

2. **Визуализация и отчеты**
   - Дашборд с ключевыми метриками
   - Круговые диаграммы по категориям
   - График cash-flow
   - Отчет о прибылях и убытках

3. **Мультипользовательский доступ**
   - Роли: Владелец, Бухгалтер, Менеджер
   - Разные уровни доступа

4. **Уведомления**
   - О предстоящих платежах
   - О низком остатке на счетах
